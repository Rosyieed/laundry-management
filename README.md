# 📌 Laundry Management System

A web application built using **ASP.NET Core MVC** for managing laundry operations such as user management, orders, and administration dashboard.

---

## 🚀 Tech Stack

- **ASP.NET Core MVC**
- **.NET SDK**
- **Entity Framework Core**
- **SQL Server**
- **AdminLTE** (Admin Dashboard Template)
- **Bootstrap**
- **Razor View Engine**
